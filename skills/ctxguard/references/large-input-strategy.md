# ctxguard large input strategy

## For large files
1. Inspect file role first
2. Read headings / top section / target ranges
3. Only full-read if required

## For multiple files
1. Inventory what was received
2. Pick the primary source of truth
3. Extract minimum needed from each file
4. Avoid duplicating the same content in output or memory

## For long tool-heavy turns
1. Restate goal
2. List next 2-4 actions
3. Batch related actions
4. Re-check latest user message before finishing
